## How to Use Answer Keys

### For Learners
1. **Complete the quiz first** — try answering all questions independently.
2. **Compare** your responses against the inline answer key (where present) or against the lesson's source material.
3. **Understand mistakes** — re-read the relevant lesson sections before retaking.
4. **Study weak areas** — the topics where you missed questions point to where to spend study time.

### For Instructors
- The in-module quizzes are the canonical source.
- Until consolidated answer keys are published, derive correct answers from the lesson content alongside each quiz.

## Academic Integrity

Answer keys (inline or planned) are for self-study and practice. Using them to complete graded assessments is academic dishonesty.

| Acceptable | Not acceptable |
|---|---|
| Checking answers after completion | Viewing answers before taking quiz |
| Studying for retakes | Copying answers for graded work |
| Understanding concepts you missed | Sharing answers during active quiz period |
| Teaching / tutoring | Helping others cheat |

## Answer-Key Template

When a per-module answer key is written, use this format:

```markdown
# Module XX: [Module Name] — Answer Key

## Question 1
**Q:** [Question text]

**Answer:** [Correct answer letter and text]

**Explanation:**
[Why this is the correct answer]

**Common Mistakes:**
- [Mistake 1 and why it's wrong]
- [Mistake 2 and why it's wrong]

**Related Material:** [Lesson reference]

---
```
