# Rubric — Thought Leadership Portfolio

This rubric is what a Principal-level reviewer (a senior who has run a similar portfolio) will use to score your work. Score yourself honestly before peer review.

Total: 100 points across 8 dimensions. Pass: ≥ 70%, with **no single dimension below 50%**. Distinction (A): 85%+, no dimension below 70%.

---

## 1. Thesis clarity and defensibility (20 pts)

The core question: do you have a thesis, or a topic area?

| Score | Description |
|---|---|
| 18–20 | One-sentence thesis (≤ 25 words), defensible with ≥ 3 concrete examples, falsifiable in principle, recognizable across all channels, anti-thesis named and addressed. A peer can recite it back. |
| 14–17 | Thesis present; supporting evidence partial; anti-thesis weak. |
| 9–13 | Thesis as topic area ("AI infra"); recitation test fails. |
| 4–8 | No specific thesis; portfolio is a content list. |
| 0–3 | No thesis. |

**Hard checks:**
- One sentence, ≤ 25 words
- 2–3 paragraph defense
- ≥ 3 concrete supporting examples from your work or industry
- Anti-thesis named with response
- Peer recitation test passed (with a real peer if possible)
- Thesis appears coherently in talk + paper + OSS arc + advisory pitch + ≥ 3 content pieces

## 2. Conference talk quality (CFP + deck + delivery readiness) (15 pts)

| Score | Description |
|---|---|
| 14–15 | CFP submitted to tier-1 venue with specific title + ≤ 200-word abstract + 3 concrete takeaways. Deck 30–45 slides drafted with ≤ 1 idea per slide. Rehearsed with peer. Accepted OR substantive reviewer feedback to revise. |
| 11–13 | CFP submitted; deck partial; not rehearsed. |
| 7–10 | CFP submitted but generic; no deck. |
| 3–6 | No CFP submitted; "I plan to..." |
| 0–2 | Not addressed. |

**Hard checks:**
- Venue chosen with defended reasoning
- CFP submitted with confirmation
- Deck drafted (30–45 slides)
- Rehearsed at least once
- Backup venue identified if rejected

## 3. Paper / essay craft (structure, evidence, voice) (15 pts)

| Score | Description |
|---|---|
| 14–15 | Paper / essay ≥ 3,500 words, structured per the hook-thesis-evidence-counter-takeaways spine, original evidence included, edited by peer, published or at editor's desk with timeline. Voice is consistent and authentic. |
| 11–13 | Paper drafted; venue chosen; editing partial. |
| 7–10 | Draft exists but generic; no peer edit. |
| 3–6 | Outline only. |
| 0–2 | Not addressed. |

**Hard checks:**
- Venue chosen with defended reasoning
- ≥ 3,500 words
- Original evidence (data, case study, novel synthesis)
- Peer-edited
- Submitted or published

## 4. OSS contribution depth (maintainer-recognized) (15 pts)

| Score | Description |
|---|---|
| 14–15 | Project chosen with thesis alignment; 6-month contribution arc planned (3+ contributions); first contribution merged with maintainer engagement on review thread; next contribution queued or in flight; demonstrated awareness of project's RFC / CONTRIBUTING process. |
| 11–13 | Project chosen; first contribution in active review; next queued. |
| 7–10 | Project chosen; one PR submitted but no maintainer interaction yet. |
| 3–6 | Project named but no contribution. |
| 0–2 | Not addressed. |

**Hard checks:**
- Project chosen with rationale (thesis alignment + maintainer culture)
- Contribution arc documented (3+ contributions over 6 months)
- First contribution merged OR in active maintainer review
- Next contribution queued with specific issue
- Project's CONTRIBUTING / RFC process understood and followed

## 5. Advisory engagement structure (legal cover + fit) (10 pts)

| Score | Description |
|---|---|
| 9–10 | 5–10 targets identified; ≥ 3 active conversations; ≥ 1 signed (or term sheet in active negotiation); employer COI cleared; deal structure (equity / cash / hybrid) defensible; time-boxing in place (≤ 8h/mo/engagement). |
| 7–8 | Targets identified; conversations active; no signed yet. |
| 4–6 | Strategy documented but no outreach. |
| 1–3 | "I'll do advisory" without specifics. |
| 0 | Not addressed. |

**Hard checks:**
- 5–10 targets identified
- ≥ 3 active conversations
- ≥ 1 signed OR term sheet
- Employer COI process followed
- Time-boxing documented

## 6. Content calendar sustainability (10 pts)

| Score | Description |
|---|---|
| 9–10 | 12-month calendar with 24+ topics laddered to thesis; ≥ 4 buffer pieces drafted; platform live with first piece published; cadence on calendar; topic mix documented (technical / opinion / response / lessons / book). |
| 7–8 | Calendar present; some buffer; platform live. |
| 4–6 | Topic list without drafted pieces. |
| 1–3 | "I'll write a newsletter" without specifics. |
| 0 | Not addressed. |

**Hard checks:**
- 12 months / 24+ topics
- ≥ 4 buffer pieces drafted
- Platform chosen + set up
- First piece published
- Cadence on personal calendar

## 7. Employer alignment & politics handled (8 pts)

| Score | Description |
|---|---|
| 7–8 | Sponsor conversation done with explicit time + approval workflow + signoff; IP boundaries documented and acknowledged; approval workflow live; sponsor update format established. |
| 5–6 | Sponsor conversation done; some details missing. |
| 3–4 | Manager aware but not formally sponsored. |
| 1–2 | "I'll handle it" without process. |
| 0 | Not addressed. |

**Hard checks:**
- Sponsor signoff captured (in writing)
- IP boundaries documented
- Approval workflow live (named contact, SLA)
- Quarterly sponsor update format established

## 8. Leverage metrics + reallocation discipline (7 pts)

| Score | Description |
|---|---|
| 6–7 | Per-channel metrics defined with collection mechanism; monthly review on calendar; first review template populated; reallocation triggers identified; baseline captured. |
| 4–5 | Metrics defined; review cadence loose. |
| 2–3 | Vanity metrics only. |
| 0–1 | Not addressed. |

**Hard checks:**
- 5 channels × leverage metrics defined
- Collection mechanism live (spreadsheet OK)
- Monthly review on calendar
- Baseline captured (Q1)

## Scoring procedure

1. **Self-score** each dimension before peer review.
2. **Peer review**: 2 personas:
   - 1 senior peer who has run a similar portfolio (the most important review)
   - 1 sponsor (your CTO / VP) for employer alignment
3. **Gap analysis**: where your self-score exceeds peer score by ≥ 3 pts, that's your blind spot. Revise.
4. **Sustained execution check**: at month 3, 6, 12, are you still operating per the cadence? The portfolio is not done at submission; it is done at month 12 with sustained output.

## Common failure modes (what costs you points)

- **The topic-area portfolio**: scoring 12/20 on thesis because you have "an interest in AI infra" instead of an argument.
- **The generic-CFP submission**: scoring 8/15 on conference talk because the abstract reads like every other abstract.
- **The opinion essay without evidence**: scoring 9/15 on paper because the original-evidence requirement is absent.
- **The drive-by OSS contribution**: scoring 8/15 on OSS because one PR doesn't make an arc.
- **The unsigned advisory pipeline**: scoring 4/10 because conversations are pleasant but no engagements actually closed.
- **The empty content calendar**: scoring 4/10 because topics exist but nothing is drafted.
- **The skipped sponsor conversation**: scoring 3/8 because the political relationship is informal — career-limiting in 6 months.
- **The vanity-metrics dashboard**: scoring 2/7 because follower count is not leverage.

## Distinction (A-grade) bar

To earn ≥ 85% you must also demonstrate:

- A **thesis that genuinely disagrees with consensus**, with the consensus position fairly represented and your disagreement defended
- A **conference talk accepted at a tier-1 venue** OR a substantive resubmission to a tier-1 venue after a thoughtful rejection
- A **paper accepted by a peer-reviewed venue** (or company engineering blog with editorial review) OR a hosted essay with documented 50k+ reads / inbound
- An **OSS contribution that becomes part of a release** with named maintainer acknowledgment
- A **signed advisory engagement with proper legal cover** that creates documented value for the founder (not just your CV)
- A **sustained cadence** — by month 12, ≥ 12 published pieces; cadence streak ≥ 6 months
- **Inbound opportunity** — by month 12, ≥ 1 substantive opportunity (job, advisory, founder conversation, conference invite) attributable to the portfolio

The discipline of building a public reputation grounded in real work, sustained over 12 months, while preserving the employer relationship and avoiding the trap of becoming a "creator," is what separates Principal-level thought leadership from very-good Senior content production.
