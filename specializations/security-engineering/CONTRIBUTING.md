# Contributing to `ai-infra-security-learning`

Thanks for helping improve this learning track.

## What belongs in this repository

- learner-facing modules in `lessons/`
- track projects in `projects/`
- assessments and rubrics in `assessments/`
- learner support material in `community/`
- shared references in `resources/`

## Repository rules

- Keep module slugs aligned with `ai-infra-security-solutions`.
- Each module should include `README.md`, lecture notes, exercises, a quiz, and resources.
- Keep validation reports, session summaries, and other operational artifacts out of the repo root. Move those to the workspace `_meta/housekeeping/` archive instead.
- Prefer runnable examples and accurate links over placeholder counts or status claims.

## Pull request checklist

- Added or updated learner-facing documentation where needed.
- Preserved naming and path consistency.
- Removed or avoided root-level housekeeping clutter.
- Verified new links and example code paths.
