# Exercise 02: Build a CLI

Build a Click-based CLI for the API from mod-002 exercise-01. Required commands:
- `ml jobs submit`, `list`, `status`, `cancel`, `retry`
- `--output json|yaml|table` everywhere
- Helpful error messages with 4-part structure (Error/Cause/Help/Docs)
- Auto-completion for bash + zsh

Bonus: telemetry shipped to a backend that aggregates command success/failure.
