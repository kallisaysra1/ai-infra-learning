### 2. Edge ML and TinyML (5 hours)
- Edge deployment patterns
- Model compression for edge
- On-device training
- Federated learning
- Edge-cloud hybrid architectures
- TensorFlow Lite, ONNX Runtime

### 3. AutoML Operations (4 hours)
- AutoML pipeline management
- Neural Architecture Search (NAS)
- Hyperparameter optimization at scale
- AutoML monitoring
- Cost management
- AutoML frameworks (AutoGluon, H2O, AutoKeras)

### 4. Real-Time ML (5 hours)
- Feature stores for real-time ML
- Online learning systems

- Low-latency serving
- Feature engineering in real-time

### 5. Emerging Patterns (4 hours)
- ML platforms and marketplaces
- Model hubs and sharing
- Continual learning systems

### 6. MLOps Maturity and Future (4 hours)
- MLOps maturity assessment
- Building MLOps teams
- Tool selection strategies
- Industry trends and predictions
- Career development in MLOps
- Contributing to MLOps community
